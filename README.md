# SZUMCMThesis

## 简介

此模板是深大杯数学建模竞赛LaTeX论文模板

## 使用方法

* 请使用XeLaTeX作为编译器
* 请使用TeX Live套装 而不要使用 CTeX套装
* 有任何问题请在[Issues](https://github.com/AirManH/SZUMCMThesis/issues)中提出
